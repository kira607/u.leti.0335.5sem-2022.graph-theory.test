\subsection*{Решение}

Оптимальние будет выбрать решение с помощью следующей леммы:

$$ P(G_1,x) = P(G,x) - P(G_2,x) $$

где
$ G_1 $ - граф, полученный из $ G $ добавлением нового ребра $ (u, v) $,
$ G_2 $ - граф, полученный стягиванием ребра $ (u, v) $ в графе $ G $.

При постепенном урощении графа получим:

$
O_{12}
- 14 * O_{11}
+ 89 * O_{10}
- 339 * O_{9}
+ 859 * O_{8}
- 1519 * O_{7}
+ 1911 * O_{6}
- 1709 * O_{5}
+ 1064 * O_{4}
- 439 * O_{3}
+ 108 * O_{2}
- 12 * O_{1}
$

Учитывая, что:
$ P(O_n, x) = x^{n} $

Получим ответ:

$
P(G, x) =
x^{12}
- 14 * x^{11}
+ 89 * x^{10}
- 339 * x^{9}
+ 859 * x^{8}
- 1519 * x^{7}
+ 1911 * x^{6}
- 1709 * x^{5}
+ 1064 * x^{4}
- 439 * x^{3}
+ 108 * x^{2}
- 12 * x
$
